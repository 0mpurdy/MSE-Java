Frequently Asked Questions (FAQ)

How can I trap error information?
  • Edit "Ministry Search Engine.lax" in your installation directory (e.g. c:\Program Files\mse) using notepad or similar text editor.
  • Change the line that begins "lax.stderr.redirect=" to "lax.stderr.redirect=c:\\mse_err.txt".
  • Change the line that begins "lax.stdout.redirect=" to "lax.stderr.redirect=c:\\mse_out.txt".
How can I increase the JVM heap size?
  • Edit "Ministry Search Engine.lax" in your installation directory (e.g. c:\Program Files\mse) using notepad or similar text editor.
  • Add "lax.nl.java.option.java.heap.size.max=512M" on a separate line at the end of the file (you should use "256M" rather than "512M" if your computer has less than 512 megabytes of memory).
