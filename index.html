Things to be aware of

The resources are not included in the Github project you will need to download or generate them separately.

You can generate the resources using the File Prepare project and the Resource seed. (More instructions on the File Prepare README)

The default location for resources is "res/" but this can be changed in the config file (generated whenever you run the project)
